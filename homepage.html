what is sorting ?

Sorting is a fundamental operation in computer science and data processing that involves arranging elements in a specific order.
Typically,the elements being sorted are stored in an array, list, or other data structure.
The goal of sorting is to make it easier to search for specific elements, access data in a specific order, or optimize algorithms that rely on data being in a certain sequence.

The most common orders for sorting are:

1. Ascending Order: Arranging elements from the smallest to the largest.
2. Descending Order: Arranging elements from the largest to the smallest.

Sorting Algorithms:

There are various sorting algorithms designed to achieve this task,
each with its own set of characteristics, advantages, and disadvantages.
Some of the most well-known sorting algorithms include:

1. bubblesort

2. Selection Sort

3. Merge Sort

4. Quick Sort

5. Heap Sort
